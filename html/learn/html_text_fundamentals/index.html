Importance

Emphasis

This word is emphasised - it has a semantic meaning, so it should not be used for making a text italic. You can use <i>something</i> instead, but you should not, because CSS should do this job.

Strong

This word is strong it has a semantic meaning, so it should not be used for making a text bold. You can use <b>something</b> instead, but you should not, because CSS should do this job
